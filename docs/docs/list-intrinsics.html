Basics

IntrinsicDescription

#alignof

Returns the alignment of the argument in bytes.

#comptime

Returns true if the argument is a compile-time constant.

#flatten

Inlines a specific call recursively, or issues a warning if it is not possible.

#inline

Inlines a specific call, or issues a warning if it is not possible.

#panic

Throws an exception in the C++ host program.

#sizeof

Returns the size of the argument in bytes.

#typeof

Returns the type of the argument.

#typename

Returns the typename of the argument (for debugging).

LLVM Analogs (libm)

The intrinsics in the table below are direct analogs for LLVM intrinsics as per the previous section. However, these are best understood specifically as analogs for floating point mathematical functions from the standard C math library.

Promotion rules

These intrinsics are overloaded for all arithmetic scalar and vector types, but all implicitly promote integral arguments to floating point.

IntrinsicLLVM Analog(s)

#sqrt

llvm.sqrt

#ceil, #floor, #round, #trunc

llvm.ceil, llvm.floor, llvm.round, llvm.trunc

#sin, #sinh, #asin, #asinh

llvm.sin, llvm.sinh, llvm.asin, llvm.asinh

#cos, #cosh, #acos, #acosh,

llvm.cos, llvm.cosh, llvm.acos, llvm.acosh

#tan, #tanh, #atan, #atanh

llvm.tan, llvm.tanh, llvm.atan, llvm.atanh

#exp, #exp2, #exp10

llvm.exp, llvm.exp2, llvm.exp10

#log, #log2, #log10

llvm.log, llvm.log2, llvm.log10

#pow

llvm.pow, llvm.powi
